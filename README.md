## Run Locally

Clone the project :

```bash
git clone https://github.com/yosuva-rajendran/react-base-setup.git
```

Go to project directory :

```bash
cd react-base-setup
```

Install dependencies :

```bash
npm install
```

Start the server :

```bash
npm run dev
```

To start db.json

```bash
 npx json-server --watch db.json
```
